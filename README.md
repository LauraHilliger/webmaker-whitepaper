# Webmaker Whitepaper -- Readme

## Installation

	git clone https://github.com/fuzzyfox/webmaker-whitepaper --depth 1\
	npm install -g grunt-cli
	npm install

### Build

	grunt build

### Develop

	npm startdev

then visit <http://localhost:1111/>

### Preview Localy

	grunt
	npm start

then visit <http://localhost:1111/>
